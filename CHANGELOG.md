# Changelog

All notable changes to **Pi Web Mobile** will be documented in this file.

The format is inspired by Keep a Changelog.

## [Unreleased]

### Added
- Mobile-first custom chat shell
- Shared auth from `~/.pi/agent/auth.json`
- Pi session history loading and continuation
- Server-side streaming transport for subscription providers
- Project picker and model picker mobile sheets
- Product documentation (`docs/`)
- Session-start context artifacts (`AGENTS.md`, `CLAUDE.md`)

### Changed
- Application renamed from experimental example to **Pi Web Mobile**

### Fixed
- Composer reset after send
- Sticky/mobile layout issues in chat shell
- Browser-side chat transport issues for subscription-based providers

## [0.1.0] - 2026-04-19

### Added
- Initial local product baseline for Pi Web Mobile
